dark-trace

A static, in-browser trace viewer that never hides overlapping events — overlapping slices on a single GPU stream are stacked into extra lanes instead of being dropped.

Credits & thanks

dark-trace's design, interaction model, and visual language are modeled directly on Perfetto and magic-trace — it would not exist without them. With gratitude to:

  • Perfetto — the trace UI this project follows: track layout, pan / zoom, selection, flow arrows, the timeline ruler, fonts, and the overall look-and-feel. google/perfetto · Apache License 2.0 · ui.perfetto.dev
  • magic-trace — interaction model and inspiration; it popularised this style of viewer. janestreet/magic-trace · MIT License · magic-trace.org
  • Chrome Trace Event Format (Google) — the JSON trace format this viewer reads.
  • PyTorch / Kineto profiler — produces the GPU traces this viewer is built for.
  • Roboto & Roboto Mono typefaces (Google) — Apache License 2.0, served via Google Fonts.

These projects are the work of their respective authors and remain under their own licenses (Apache-2.0 / MIT). Please support and credit them too.
